The definition of private... in some way.

PornHubOS (also known as PornAOSP or PAOSP) is an Android distribution focused privacy when doing... eeeem... certain things.

PornAOSP is designed with simplicity and user experience in mind. Based on LineageOS, it's build the finest stability and greatest features to the user.

Focused on improving what's greatest in Android and not sophisticating things, PornAOSP provides smooth user experience with great customization options left to end user.

Wait... what?

It looks fake to you, right? Well, you're wrong. It's actually real. And you can see the sources here.

This project started as a joke on a Polish Redmi Note 4 users group (not availible anyway). One guy managed to post a fake screenshot of it running to @rn4official and it was greeted with great enthusiasm, and even greater expectations. So, while it was getting viral, we thought "Why not make it an actual ROM?". That's how the whole story started.

Our Telegram group built instantly, and grew to 250 members within a few hours. It also managed to run through a few of attacks within that period, but it's a longer story :v

After some time we considered the fact of using PornHub trademark. We managed to release one build with PornHubOS name, but later ones will be branded PornAOSP. We strongly believe that name is much more neutral. We don't want to risk a copyright abuse. However, the final name may change.

Features

  • Based on Android Nougat and Oreo
  • Substratum support
  • A PornHub-inspired theme
  • LegitBrowser - a browser automatically clearing "sensitive" history
  • PORN - Probably Ordinary Recovery Nonsense
  • Pornograph - an audio player based on Phonograph
  • Pre-rooted with Magisk
  • F-Droid and AdAway included
  • Lawnchair as default launcher
  • Custom wallpapers
  • A ZIP with some apps for your pleasure ( ͡° ͜ʖ ͡°)

OK... When?

We can't tell you the exact ETA. We don't know when the first build will be finished.

This is a for-fun project and we don't care about any deadlines. We're doing what we want to do and experimenting with the project.

The first build will be released for Xiaomi Redmi Note 4 Snapdragon (mido). We're also planning to release it for Mi 4 (cancro), but a build for it will be released much later. We're not planning any other official ports right now.

Please don't ask us for ETAs. It'll be released when we think it's stable enough. We can promise you that every ETA question makes the project release date postpone. Consider that.
